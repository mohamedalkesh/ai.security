دليل واجهات منصة الأمن السيبراني بالعربي

1️⃣ صفحة تسجيل الدخول (login.html)

الغرض: الدخول إلى المنصة باستخدام بيانات الاعتماد

العناصر:

  • Username: اسم المستخدم — أدخل اسم المستخدم الخاص بك
  • Password: كلمة المرور — أدخل كلمة المرور (يمكنك إظهارها/إخفاءها بالزر 👁️)
  • Role: الدور — اختر صلاحيتك:
  • Administrator — مسؤول النظام (يستطيع رؤية وإدارة كل شيء)
  • Organisation Admin — مسؤول المنظمة (يدير مستخدمي منظمته فقط)
  • Security Analyst — محلل أمني (يحق له التحقيق في التنبيهات)
  • Viewer — مشاهد (قراءة فقط، لا تعديل)
  • Remember me: تذكرني — يحفظ الجلسة في المتصفح
  • Forgot Password?: نسيت كلمة المرور — يوجهك لصفحة الاستعادة
  • Sign In: زر الدخول — يرسل البيانات للسيرفر للتحقق
  • Request enterprise access: طلب وصول مؤسسي — يفتح نموذج لطلب إنشاء حساب مؤسسي

2️⃣ لوحة التحكم (dashboard.html)

الغرض: نظرة عامة على حالة الأمان في الوقت الحقيقي

القسم العلوي:

  • Search: بحث — ابحث عن حوادث، عناوين IP، مستخدمين
  • Notifications 🔔: الإشعارات — يعرض قائمة الإشعارات الجديدة
  • Mark all read — عيّن الكل كمقروء
  • View all alerts → — انتقل لصفحة التنبيهات
  • User Chip: معلومات المستخدم — اسمك ودورك في النظام

الإحصائيات (KPIs):

  • Active Alerts 🔴: عدد التنبيهات النشطة حالياً
  • Threats Blocked 🟢: عدد التهديدات التي تم حظرها
  • System Health 📊: صحة النظام كنسبة مئوية
  • Avg Response Time ⏱️: متوسط وقت الاستجابة بالثواني

الرسوم البيانية:

  • Threat Activity: نشاط التهديدات — رسم بياني للهجمات والحجب خلال فترة زمنية
  • أزرار: Last 24 Hours / 7 Days / 30 Days — تغيير نطاق الوقت
  • Attack Types: أنواع الهجمات — رسم دائري يوزع الهجمات حسب النوع

الأسفل:

  • Recent Alerts: أحدث التنبيهات
  • Activity Log: سجل النشاط

3️⃣ صفحة التنبيهات (alerts.html)

الغرض: إدارة والتحقيق في جميع تنبيهات الأمان

الشريط العلوي:

  • Severity Pills: حبوب الخطورة — عدد التنبيهات لكل مستوى:
  • Critical 🔴 — حرجة
  • High 🟠 — عالية
  • Medium 🟡 — متوسطة
  • Low 🔵 — منخفضة

شريط الأدوات:

  • Search: بحث — بالمعرف، النوع، أو عنوان IP
  • Filter by Severity: تصفية حسب الخطورة
  • Sort by: ترتيب حسب — الوقت / الخطورة / المعرف
  • Export CSV 📥: تصدير قائمة التنبيهات
  • Toggle Columns 🎛️: إظهار/إخفاء أعمدة الجدول

شريط الإجراءات الجماعية:

  • Investigate 👁️: عيّن كـ "قيد التحقيق"
  • Resolve ✅: عيّن كـ "محلولة"
  • False Positive 👎: عيّن كإيجابية خاطئة
  • Clear ❌: إلغاء الاختيار

الجدول:

الأعمدة: INCIDENT ID, TYPE, SEVERITY, SOURCE, STATUS, ASSIGNED, TIME, ACTIONS

نافذة التفاصيل (Modal):

  • Assigned To: تعيين لمحلل آخر
  • ML Feedback: تغذية راجعة للذكاء الاصطناعي:
  • True Attack 👍 — هجوم حقيقي
  • False Positive 👎 — إيجابية خاطئة
  • Uncertain ❓ — غير مؤكد
  • Investigation Notes: ملاحظات التحقيق
  • أزرار الإجراء: Investigating, False Positive, Mark Resolved

4️⃣ صفحة مسح PCAP (scan.html)

الغرض: رفع ملفات PCAP للتحليل باستخدام الذكاء الاصطناعي

منطقة الرفع:

  • Drop PCAP file here: اسحب وأفلت ملف PCAP هنا
  • Browse File: تصفح لاختيار ملف
  • Formats: .pcap, .pcapng, .cap
  • Max size: 100 MB

شريط التقدم:

  • Uploading ☁️: رفع الملف للسيرفر
  • AI Analyzing 🧠: استخراج التدفقات وتصنيفها باستخدام XGBoost
  • Saving Alerts 💾: حفظ التنبيهات

النتائج:

  • Total Flows: إجمالي عدد التدفقات
  • Benign Flows 🟢: التدفقات السليمة
  • Attack Flows 🔴: التدفقات الخبيثة
  • Avg Confidence: متوسط ثقة النموذج

جدول الهجمات المكتشفة:

Attack Type, Count, Severity, Sample Source

أزرار:

  • Scan Another File 🔄: مسح ملف جديد
  • Go to Alerts ⚠️: الانتقال لصفحة التنبيهات

5️⃣ صفحة الحوادث (incidents.html)

الغرض: عرض الحوادث (مجموعات من التنبيهات ذات الصلة)

الشريط العلوي:

  • Open: عدد الحوادث المفتوحة
  • Resolved: عدد الحوادث المحلولة

شريط الأدوات:

  • Search: بحث بالعنوان أو IP المصدر
  • Filter by Status: تصفية حسب الحالة
  • Refresh 🔄: تحديث القائمة

بطاقة الحادث:

Title, Source IP, Attack Family, Alert Count, Status, Time

6️⃣ صفحة التحليلات (analytics.html)

الغرض: عرض مؤشرات الأداء التشغيلية والاتجاهات

نطاق الوقت:

Last 7 days / 14 days / 30 days / 90 days

مؤشرات الأداء (KPIs):

  • Mean Time to Resolve (MTTR): متوسط وقت حل التنبيهات
  • Total in Window: إجمالي التنبيهات
  • Peak Day: اليوم الأكثر ازدحاماً
  • Unique Attackers: عدد عناوين IP المهاجمة الفريدة

الرسوم البيانية:

  • Daily Alert Volume: حجم التنبيهات اليومي
  • Severity Trend: اتجاه الخطورة

القسم السفلي:

  • Top Attackers 🔥: أكثر عناوين IP هجوماً

7️⃣ صفحة Webhooks (webhooks.html)

الغرض: إدارة تكاملات التنبيهات الخارجية

الشريط العلوي:

  • New Webhook ➕: إنشاء تكامل جديد

بطاقة Webhook:

Name, URL, Preset, Min Severity, Status أزرار: Edit, Rotate Secret, Delete

نافذة الإنشاء/التعديل:

Name, Endpoint URL, Preset, Minimum Severity, Enabled, Secret (HMAC-SHA256)

8️⃣ صفحة سجل التدقيق (audit.html)

الغرض: سجل ثابت لكل الإجراءات الحساسة

الشريط العلوي:

Login OK, Login Fail, Rate Limit counts

شريط الأدوات:

Search, Filter by Action (LOGIN_OK, LOGIN_FAIL, REGISTER_OK, RATE_LIMIT, ALERT_STATUS, ALERT_UPDATE, ALERT_DELETE, ALERT_BULK, ALERT_COMMENT), Export CSV, Refresh

الجدول:

WHEN, ACTION, ACTOR, RESOURCE, SOURCE IP, DETAILS

9️⃣ صفحة المراقبة (monitoring.html)

الغرض: مراقبة صحة النظام والأداء في الوقت الحقيقي

بطاقات المؤشرات:

  • CPU Usage 🖥️: استخدام المعالج
  • Memory Usage 💾: استخدام الذاكرة
  • Network Traffic 🌐: حركة الشبكة
  • Disk Usage 💿: استخدام القرص

الرسوم البيانية:

CPU Usage — Last 24h, Network Traffic — Last 24h

شاشة مراقبة الشبكة الحية:

Network Interface, Start/Stop buttons مؤشرات: Total Flows, Benign, Attacks, Flows/sec, Attacks/sec, Queue Depth Traffic last 60s sparkline, Recent Detections

جدول حالة الخدمات:

SERVICE NAME, STATUS, UPTIME, CPU, MEMORY, ACTIONS (Restart / View Logs)

نافذة السجلات:

Download, Auto-scroll ألوان: INFO (أزرق), WARN (أصفر), ERROR (أحمر), DEBUG (بنفسجي)

🔟 صفحة التقارير (reports.html)

الغرض: إنشاء وتصدير تقارير الأمان

الشريط العلوي:

Export CSV, Generate PDF Report

مؤشرات الأداء:

Total Incidents, Block Rate, Avg Response Time, Active Threats

الرسوم البيانية:

Incident Trends (4/6/12 months), Threat Distribution (donut)

جدول التقارير الأخيرة:

REPORT NAME, TYPE, DATE, SIZE, ACTIONS

1️⃣4️⃣ صفحة الاتصال (contact.html)

الغرض: التواصل مع فريق الدعم

النموذج:

Subject, Message, Send

🎯 ملخص الصلاحيات حسب الدور

الدور يمكنه رؤية يمكنه تعديل صفحات خاصة
ADMIN كل شيء (System tenant only) كل شيء Settings, Audit, Webhooks
ORG_ADMIN بيانات منظمته فقط مستخدمي منظمته Audit, Webhooks
ANALYST بيانات منظمته فقط التنبيهات والحوادث لا شيء خاص
VIEWER بيانات منظمته فقط لا شيء (قراءة فقط) لا شيء خاص
